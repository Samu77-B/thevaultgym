# Documentation and Unused Files

This folder contains unused files and documentation that have been moved from the main website directory.

## Unused Files Moved Here

### JavaScript Files
- **add-footer.js** - Legacy footer script that was replaced by `include-footer.js` and `components/footer.html`. This file is no longer referenced in any HTML files.
- **update-all-pages.js** - Node.js utility script used for one-time updates to service pages. Not needed for production.

### CSS Files
- **footer.css** - Unused CSS file. The website uses `footer-styles.css` instead.

### PHP Files
- **default.php** - Default Hostinger hosting page. Not part of the actual website.

## Notes

- These files are kept for reference purposes only
- They are not loaded or executed by the website
- They can be safely deleted if disk space is a concern
